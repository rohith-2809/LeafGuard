# üåø LeafGuard ‚Äì AI-Powered Plant Disease Detection & Treatment Intelligence System

LeafGuard is a fully engineered AI platform built to analyze plant leaf images, detect diseases with high precision, and deliver scientifically grounded treatment recommendations.
It integrates computer vision, deep learning, multi-service distributed architecture, and Gemini-based reasoning into one cohesive, production-ready ecosystem.

Designed for agritech platforms, research labs, greenhouse automation, and intelligent farming tools.

## üöÄ Tech Stack (Deep Engineering View)

### üß† AI & Machine Learning

* ResNet50 backbone fine-tuned using a two-phase transfer learning pipeline
* 14,000-image dataset spanning multi-class leaf conditions
* Gradient-stabilized optimization using AdamW with decay scheduling
* Domain-focused augmentation pipeline including color space distortion, contrast jitter, and geometric perturbations
* Softmax-based multi-class prediction head for disease classification
* Fully quantization-tolerant architecture for edge deployment (optional)
* Hugging Face Inference Endpoints for distributed cloud prediction

* Non-blocking asynchronous prediction flow with UI state isolation

### üîß Backend Architecture

* Node.js + Express acting as an API gateway and service coordinator
* Microservice separation between inference, reasoning, and data persistence
* Secure request isolation for synchronous and asynchronous prediction cycles
* Data contracts for standardized image and prediction payloads
